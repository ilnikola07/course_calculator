****Описание проекта****


Данный репозиторий содержит программное решение для курсовой работы по дисциплине "Алгоритмизация и программирование". 


Проект реализован с использованием архитектуры, разделяющей пользовательский интерфейс и бизнес-логику, что обеспечивает модульность и удобство тестирования.


**Основные возможности**


Графический интерфейс (WinForms): максимально простой внешний вид, соответствующий требованиям простого десктопного калькулятора.


Разделение логики: "внутренности калькулятора" вынесены в отдельный класс CalcEngine. Работа основана на последовательной работе трёх алгоритмов: 
1. Токенизации
2. Сортировочной станции
3. Расчёт в ОПЗ


Именно благодаря этому вам становятся доступны:


Простые операции сложения/вычитания/умножения/деления/возведения в степень/извлечения корня


Возможность высчитать сложные математические функции (sin, cos, tg, log и пр.)


Возможность задавать собственные переменные и использовать их в вычислениях


Помимо основного вычислительного класса создан CalcHistory, позволяющий хранить историю ваших вычислений прямо в приложении!



**Технологический стек**


Язык программирования: C# версия 12.0


Платформа: .NET Framework 4.8.1


UI: Windows Forms (WinForms)


Архитектура: Class Library (выделение логики в отдельный проект CalcLib)


Тестирование: xUnit


**Установка и настройка**


1. Получение исходного кода. Склонируйте репозиторий или загрузите архив с исходным кодом по ссылке https://github.com/ilnikola07/course_calculator.git


2. Открытие проекта. Запустите Visual Studio (рекомендуется последняя стабильная версия).
Откройте файл решения (.sln), расположенный в папке проекта.


3. Сборка. Выполните команду Build Solution (Собрать решение) для компиляции всех компонентов. При возникновении ошибок сборки проверьте целостность кода и конфигурацию проекта, при надобности просто закомментируйте ошибочные строки внутри странных файлов. Всё готово к использованию!


**Использование**


Запустите проект (клавиша F5 или кнопка Start).
В открывшемся окне WinForms вы увидите все доступные возможности. 


При нажатии "Показать историю", у вас появится форма, в которой будет отображаться ваша история. При надобности, кликните по старому выражению и оно у вас сразу появится в текущей строке ввода. Совершите расчёт снова! 


При нажатии "Что это такое?", вы можете узнать описание этого проекта.



**Тестирование**


Проект покрыт модульными тестами с использованием xUnit.
Откройте Test Explorer (Тестовый обозреватель) в Visual Studio (Test -> Windows -> Test Explorer).
Нажмите Run All Tests (Запустить все тесты).
Убедитесь, что все тесты пройдены успешно (отображены зелеными маркерами - должно быть 69).
